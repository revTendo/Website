How to connect to revTendo?

Requirements:
• a working Nintendo 3DS / 2DS (emulators are not supported)
• a working internet connection
• credentials from https://pine-cone.xyz/reBadge

How to connect:

  • Open the System Settings on your 3DS
  • Go to Internet Settings
  • Go to Connection Settings
  • Choose a connection and select Change Settings
  • Change the DNS settings to Manual
  • Set the Primary DNS to and port to
  • Enable Basic Authentication
  • Enter your credentials from https://pine-cone.xyz/reBadge

You need to be in the Discord server for security reasons.
